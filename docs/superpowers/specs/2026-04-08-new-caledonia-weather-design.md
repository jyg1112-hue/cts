# 뉴칼레도니아 날씨 현황 위젯 — 디자인 스펙

## Context

항구에서 하역하는 니켈 광석의 원산지가 뉴칼레도니아이므로, 현지 주요 광산의 날씨 현황을 하역 데이터 대시보드에서 바로 확인할 수 있도록 한다. 별도 페이지 이동 없이 대시보드 내 위젯으로 제공한다.

---

## 위치

`unloading_data.html` — `.main-grid` (3-column) 의 세 번째 칸에 기존 패널과 동일한 구조로 삽입.

---

## 표시 데이터

| 항목 | 내용 |
|------|------|
| 현재 날씨 | 기온(°C), 날씨 상태 이모지 (맑음/구름/비/뇌우 등) |
| 예보 | 오늘 포함 3일치 (기온 + 날씨 이모지) |
| 제외 | 풍속, 풍향 표시 없음 |

## 날씨 API

**Open-Meteo** (`https://api.open-meteo.com/v1/forecast`)
- 무료, API 키 불필요
- 파라미터: `latitude`, `longitude`, `current=temperature_2m,weathercode`, `daily=temperature_2m_max,weathercode`, `forecast_days=3`, `timezone=Pacific/Noumea`
- WMO 날씨 코드 → 이모지 매핑 (0=☀️, 1-3=⛅, 51-67=🌧️, 71-77=🌨️, 80-82=🌦️, 95-99=⛈️)

## UI 구성

### 패널 구조

```
┌─────────────────────────────────────┐
│ 뉴칼레도니아 날씨 현황    [● 실시간] │  ← weather-header
├─────────────────────────────────────┤
│                                     │
│   [SVG 지도 + 마커 + 툴팁]          │  ← map-area (flex: 1)
│                                     │
├─────────────────────────────────────┤
│ ● Open-Meteo · 자동 갱신  방금 업데이트됨 │  ← weather-footer
└─────────────────────────────────────┘
```

### 색상 (대시보드 변수 활용)

| 요소 | 색상 |
|------|------|
| 패널 배경 | `#ffffff` |
| 패널 테두리 | `#e5e7eb` |
| 바다 (SVG 배경) | `#dbeafe` → `#bfdbfe` (그라데이션) |
| 섬 (육지) | `#d1fae5` → `#a7f3d0` |
| 기본 마커 | `#1a3a5c` (대시보드 헤더 컬러) |
| 호버 마커 | `#3b82f6` (대시보드 블루) |
| 툴팁 | 흰 카드, `box-shadow: 0 4px 16px rgba(0,0,0,0.12)` |
| 실시간 뱃지 | `#ecfdf5` 배경, `#10b981` 텍스트/dot |

### 인터랙션

- 마커 위에 기온 + 날씨 이모지 항상 표시
- 마커 hover → 3일 예보 툴팁 (위치: 마커 상단, 패널 경계 벗어나지 않게 좌우 clamp)
- hover 마커는 파란색(`#3b82f6`)으로 강조, 나머지는 네이비(`#1a3a5c`)
- 툴팁: 광산명 / 오늘·내일·모레 행 (이모지 + 날씨명 + 기온)

### 자동 갱신

- 페이지 로드 시 즉시 fetch
- `setInterval` 30분마다 재fetch
- footer에 마지막 업데이트 시각 표시 (`HH:MM` 형식)

## 파일 변경 범위

| 파일 | 변경 내용 |
|------|-----------|
| `unloading_data.html` | `.main-grid` 세 번째 칸에 날씨 패널 HTML 추가, 인라인 CSS 추가, JS fetch 로직 추가 |

백엔드 변경 없음. Open-Meteo는 브라우저에서 직접 호출.

---

## 검증 방법

1. `http://localhost:8000/unloading-data` (또는 파일 직접 열기) 접속
2. 날씨 패널이 세 번째 칸에 정상 렌더링되는지 확인
3. 각 마커에 기온과 날씨 이모지가 표시되는지 확인
4. 마커 hover 시 3일 예보 툴팁이 나타나는지 확인
5. 브라우저 네트워크 탭에서 Open-Meteo API 호출 5건 성공 확인
6. 30분 후 자동 갱신 및 footer 시각 갱신 확인

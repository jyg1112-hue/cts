# 야드 시뮬레이션 페이지 — 설계 문서

**날짜:** 2026-04-11  
**파일:** `yard.html` (기존 파일 전면 교체)

---

## 1. 목표

기존 `yard.html`을 개선하여:
- 선석 스케줄(`berth7_v6` localStorage)의 석탄 반입 데이터를 자동으로 불러와 시뮬레이션 테이블을 채운다.
- 자가야드 / 임차야드 두 개 야드의 재고·적치율을 날짜별로 시뮬레이션한다.
- 반출은 수동 입력(고정 t/일 또는 날짜별 직접 입력)으로 처리한다.

---

## 2. 데이터 소스

### 반입 (읽기 전용)
- **소스:** `localStorage.getItem('berth7_v6')` (schedule.html이 저장)
- **필터:** `item.type === 'ship' && item.cargoType === '석탄'`
- **반입량:** `item.bl` (톤) — 없으면 기본값 사용
- **날짜 배분:** `item.start` ~ `item.end` 기간에 시간 비율로 균등 분산 (기존 yard.html 방식 유지)
- **야드 배정:** 기본값 → 자가야드. 임차야드 반입은 별도 수동 설정 가능

### 반출 (수동 입력)
- `berth7_v6`, `banchu_v1` 연동 없음
- 야드별로 이적·육송·해송 반출(t/일) 직접 입력
- 입력 방식: **일 고정** (전 기간 동일값) 또는 **일별 직접 입력** (날짜별 개별값)
- localStorage에 저장: `yard_sim_v1` 키

---

## 3. 화면 구성

### 3-1. 헤더 툴바
```
[반입 데이터 불러오기]  [상세 설정]  |  마지막 업데이트: HH:MM
자가야드 XXX,XXXt 50.1%  |  임차야드 XXX,XXXt 39.5%  |  총재고 XXX,XXXt
```
- "반입 데이터 불러오기" 클릭 → `berth7_v6` 읽어 테이블 반입 열 갱신
- 기존 수동 입력값이 있으면 "덮어쓰겠습니까?" 확인 모달 표시

### 3-2. 메인 시뮬레이션 테이블 (오늘 ~ 60일, 가로 스크롤)

| 날짜 | 자가야드 반입 | 자가야드 반출 | 자가야드 재고 | 자가야드 적치율 | 임차야드 반입 | 임차야드 반출 | 임차야드 재고 | 임차야드 적치율 | 전체 반입 | 전체 반출 | 총재고 |
|------|------|------|------|------|------|------|------|------|------|------|------|

**적치율 색상:**
- ~80%: 기본색 (검정)
- 80~90%: `#d97706` (노랑)
- 90~100%: `#ea580c` (주황)
- 100%+: `#dc2626` (빨강)

**셀 배경:** 흰색 (자동/수동 구분 없이 동일)  
**수동 편집:** 셀 클릭 → 인라인 편집, 편집된 셀은 파란 테두리(`border: 2px solid #3b82f6`)로 표시

### 3-3. 상세 설정 패널 (모달)

야드별(자가야드 / 임차야드) 각각:
- 최대 용량 (t)
- 초기 재고 (t)
- 반출 입력 방식 탭:
  - **일 고정**: 이적·육송·해송 t/일 입력 → 반출합 자동 계산
  - **일별 입력**: 날짜 × 이적·육송·해송 테이블 (60일치)

Silo (참고용, 시뮬레이션 제외):
- 현재 재고 수치만 입력 (표시용)

저장 시 `yard_sim_v1` localStorage에 기록 후 테이블 즉시 재계산.

## 6. 계산 로직

```
각 날짜 d에 대해:
  자가야드_반입[d] = 자동반입(berth7_v6에서 계산) + 수동반입_자가[d]
  자가야드_반출[d] = outMode === 'fixed' ? outFixed 합 : outDaily[d] 합
  자가야드_재고[d] = 자가야드_재고[d-1] + 자가야드_반입[d] - 자가야드_반출[d]
  자가야드_적치율[d] = 자가야드_재고[d] / selfYard.capa * 100

  임차야드도 동일 로직
  총재고[d] = 자가야드_재고[d] + 임차야드_재고[d]
```

## 7. 제약 사항

- `berth7_v6`, `banchu_v1` 데이터는 읽기 전용 (기존 페이지 변경 금지)
- 저장은 "상세 설정 > 저장·재계산" 버튼으로만
- Silo는 시뮬레이션 계산에서 제외, 참고 표시만
